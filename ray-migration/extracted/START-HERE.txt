============================================================
 Ray Plumbing WEBSITE — SOURCE CODE
============================================================

This folder contains the complete, editable source code for the
Ray Plumbing website. Any web developer can run, edit, and deploy it.
It does NOT depend on any outside account or service.

------------------------------------------------------------
 WHAT YOU NEED
------------------------------------------------------------
- Node.js (free): https://nodejs.org  (install the "LTS" version)

------------------------------------------------------------
 TO PREVIEW / EDIT THE SITE
------------------------------------------------------------
1. Open a terminal in this folder.
2. Run:   npm install        (one time, downloads libraries)
3. Run:   npm run dev         (starts a local preview)
4. Open the http://localhost link it prints in a browser.

Edit any file inside the "src" folder and the preview updates
live. Images live in "public/images".

NOTE: The local preview (npm run dev) is intentionally slower
than the real site -- it is for editing. The deployed site is
fully optimized and fast.

------------------------------------------------------------
 TO DEPLOY THE SITE (put it live)
------------------------------------------------------------
1. Run:   npm run build
2. This creates a "dist" folder = the finished website.
3. Host it anywhere that serves static files:
      - Netlify: drag the "dist" folder onto the dashboard
      - Or Vercel, Cloudflare Pages, GitHub Pages, etc.

IMPORTANT: The file public/_redirects is included so page
links work correctly on Netlify. Keep it.

------------------------------------------------------------
 GOOD TO KNOW
------------------------------------------------------------
- All images are stored locally in this project. Nothing loads
  from an external account, so the site keeps working forever.
- "node_modules" and "dist" are NOT included on purpose -- they
  are regenerated by the two commands above.

See README.md for more detail.
